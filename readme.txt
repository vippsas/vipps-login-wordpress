== Description ==
With this plugin, you can log into your Wordpress installation or your Woocommerce store using your Vipps app - no passwords

== Installation ==
1.  Install the plugin using WordPressâ€™ [built-in installer](https://codex.wordpress.org/Managing_Plugins#Installing_Plugins). The plugin can also be installed manually by upload the plugin files to the /wp-content/plugins/ directory.
2. Activate the plugin through the \'Plugins\' screen in WordPress.
3. From the Vipps portal (portal.vipps.no), get your Client ID and Client Secret and add them to the Settings-page for this plugin
4. Note your callback URL form the settings-page and add that to the Vipps Portal
5. Configure your plugin as needed to allow logging in using your Vipps app.
